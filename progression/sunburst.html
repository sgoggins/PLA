Select a Course/Major combination from the dropdown. PHYSICS.Physics.TERM.csv is a good start. This approximates similar plots we've made with real data. Then mouse-over the diagram to explore the paths students take through a set of courses. The innermost ring represents courses in the SUBJECT taken by first term students, the next ring represents the second term, etc. This surveys all courses within the subject included in the PLA-MOOC data set. The percentage of students in this major following this path is given in the middle. The courses along this path light up on the right side on the breadcrumb trail.

  1. Percentages represent the total number of ALL majors that followed this path.
  2. A slash separates multiple courses taken in the same term.
  3. "NONE" indicates students that took no course in this subject in a particular term.
  4. We allow for up to 12 terms (12 rings), but because the PLA-MOOC data contains only large courses, most of these will be "NONE".
